# Reko

Emotion recognition Android application. My graduate work 👨‍🎓

## Stack

- TensorFlow Lite
- CameraX
- RxJava 2
- Dagger 2
- Retrofit
- Firebase

## Emotion recognition

Emotions can be recognized with:
- Local model ([sources](https://github.com/weazyexe/reko-model))
- Skybiometry recognition API
